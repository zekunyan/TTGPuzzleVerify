# TTGPuzzleVerify Examples

The examples are intentionally split into two independent projects so Objective-C and Swift integrations can be opened, installed, and tested separately.

## Objective-C UIKit example

```sh
cd Examples/ObjCExample
pod install
open TTGPuzzleVerify.xcworkspace
```

This project contains the Objective-C UIKit demo screens and the Objective-C XCTest target.

## Swift example

```sh
cd Examples/SwiftExample
pod install
open TTGPuzzleVerifySwiftExample.xcworkspace
```

This project contains the Swift UIKit demo and the SwiftUI `UIViewRepresentable` demo.

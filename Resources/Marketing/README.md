# Marketing HTML Assets

This folder contains editable HTML marketing assets for TTGPuzzleVerify.

- `promo_poster.html` -> `promo_poster.jpg`: 16:9 promotional poster highlighting the project value proposition, feature set, and integrations.
- `architecture_diagram.html` -> `architecture_diagram.jpg`: architecture overview showing integration surfaces, the shared core, feature modules, and verification flow.
- `quick_start.html` -> `quick_start.jpg`: README quick start visual built from real Swift and Objective-C example screenshots.

The `assets/` folder contains screenshots captured from the runnable examples. Re-run the examples, replace those files with fresh screenshots, then open the HTML in a browser and export the final JPG/PNG assets.

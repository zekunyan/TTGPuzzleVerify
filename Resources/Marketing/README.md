# Marketing HTML Assets

This folder contains editable HTML marketing assets for TTGPuzzleVerify.

- `promo_poster.html`: 16:9 promotional poster highlighting the project value proposition, feature set, and integrations.
- `architecture_diagram.html`: architecture overview showing consumers, core modules, distribution, and the verification flow.

Both files are self-contained HTML/CSS documents. Open them in a browser and use the browser print/export tools to produce PNG or PDF versions.

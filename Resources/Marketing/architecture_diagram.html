Architecture overview

TTGPuzzleVerify
Swift Core

A Swift 5.9 verification view designed around rendering, interaction, metrics, state, and clean UIKit / SwiftUI / Objective-C integration.

Consumers

SSwiftUI Demo

Uses UIViewRepresentable bindings for verification state, progress, result summary, and callbacks.

USwift UIKit

Direct view integration with configuration, horizontal mode, failure feedback, and result labels.

OObjective-C UIKit

Imports the generated Swift header and uses bridged enums, delegate methods, and blocks.

TTGPuzzleVerifyView

UIKit UIView · @objc API

Rendering Layer

  • Back image mask
  • Front hole mask
  • Puzzle piece mask
  • Inner and piece shadows

Pattern Engine

  • Classic puzzle path
  • Square / circle paths
  • Custom UIBezierPath
  • Scaled and translated masks

Interaction Engine

  • Pan gesture handling
  • Horizontal / vertical / both axes
  • Percentage setters
  • Clamped bounds

Verification State

  • Idle / dragging
  • Verified / failed
  • Retry and locked states
  • Auto-snap success

Metrics & Result

  • Track points
  • Timestamp and velocity
  • Elapsed time
  • Offset and drag distance

Configuration API

  • Behavior configuration
  • Style object
  • Delegate callbacks
  • Swift closures

Distribution

PCocoaPods

iOS 16.0 deployment target, Swift 5.9 metadata, local path examples.

MSwift Package

SPM manifest exposes the Swift source as the TTGPuzzleVerify library product.

TTests
